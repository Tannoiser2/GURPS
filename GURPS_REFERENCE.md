## Note meccaniche GURPS Lite

- **Tiro di successo:** 3d6 ≤ livello skill. Critico se ≤ 4 o ≤ livello−10. Fallimento critico se ≥ 18 o ≥ livello+10.
- **Default:** se non si ha la skill, si tira a (attributo − penalità). E.g. combattere default = FO−5.
- **Schivata base:** DE/2 + 3 (arrotonda giù).
- **Basic Speed:** (DE + SA) / 4.
- **PF base:** uguale a SA. Feriti se PF < max/3. Fuori combattimento se PF ≤ 0.
- **Colpo critico:** danni massimi + nessuna difesa avversaria.
- **Tiro morte:** 3d6 ≤ SA quando PF ≤ −PF (o −2×PF con Duro da Uccidere).
